⚙️ Setup Guide — IT Borrow System

ทำตาม 5 ขั้นตอน ใช้เวลาประมาณ 15 นาที

1 สร้าง Google Apps Script

  1. เปิด Google Sheet → เมนู Extensions → Apps Script
  2. ลบ code เดิม แล้ววาง code จากไฟล์ apps-script.gs
  3. บันทึก (Ctrl+S)

2 สร้าง Sheet tabs (รันครั้งเดียว)

  1. ใน Apps Script กด Run → initSheets
  2. อนุญาต permission ที่ขอ (Google จะถาม)
  3. จะสร้าง 5 tabs อัตโนมัติ: employees, equipment, requests, request_items, stock_log
⚠️ ถ้า tab ชื่อ การตอบสนองของแบบฟอร์ม 1 มีอยู่แล้ว ไม่ต้องลบ — ระบบใหม่ใช้ tabs แยกต่างหาก

3 กรอกข้อมูลพนักงาน

เปิด tab employees แล้วกรอกข้อมูล:

role ต้องเป็น admin หรือ user ตัวพิมพ์เล็กเท่านั้น

4 Deploy Apps Script เป็น Web App

  1. กด Deploy → New deployment
  2. Type: Web app
  3. Execute as: Me
  4. Who has access: Anyone
  5. กด Deploy → Copy Web app URL
⚠️ ทุกครั้งที่แก้ code ต้อง Deploy ใหม่ → Manage deployments → Edit → New version

5 ใส่ URL ใน index.html แล้ว Push ขึ้น GitHub

เปิด index.html แก้ไขบรรทัดนี้:

var API_URL  = 'YOUR_APPS_SCRIPT_WEB_APP_URL'; // ← วาง URL จาก Step 4
  1. Push ไฟล์ index.html ขึ้น github.com/naadisak/itborrow
  2. Settings → Pages → Source: main / root → Save
  3. รอ 1-2 นาที → เข้าใช้งานที่ naadisak.github.io/itborrow
