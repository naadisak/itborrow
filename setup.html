⚙️ คู่มือติดตั้งระบบยืม-คืนอุปกรณ์ไอที

ทำตาม 4 ขั้นตอนนี้ ใช้เวลาประมาณ 10 นาที

1 สร้าง Google Apps Script

  1. เปิด Google Sheet ที่เก็บข้อมูลการยืม
  2. คลิกเมนู Extensions → Apps Script
  3. ลบ code เดิมออกทั้งหมด แล้ว วาง code จากไฟล์ apps-script.gs
  4. ตรวจสอบตัวแปรนี้ว่าตรงกับชื่อ tab ของ Sheet:
var SHEET_NAME = 'การตอบสนองของแบบฟอร์ม 1'; // ← แก้ให้ตรงกับชื่อ tab
💡 ดูชื่อ tab ได้ที่แถบด้านล่างของ Google Sheet

2 Deploy Apps Script เป็น Web App

  1. กด Deploy → New deployment
  2. Type: เลือก Web app
  3. Execute as: Me
  4. Who has access: Anyone
  5. กด Deploy แล้ว Copy Web app URL
⚠️ ถ้าแก้ไข code แล้ว ต้อง Deploy ใหม่ทุกครั้ง (Deploy → Manage deployments → Edit → New version)

3 ใส่ URL ใน index.html

เปิดไฟล์ index.html แก้ไขส่วนนี้:

var APPS_SCRIPT_URL = 'YOUR_APPS_SCRIPT_WEB_APP_URL'; // ← วาง URL จาก Step 2 var GOOGLE_FORM_URL = 'YOUR_GOOGLE_FORM_URL'; // ← URL Google Form ยืมอุปกรณ์
💡 URL Google Form หาได้จาก Google Form → Send → ไอคอน link 🔗

4 Push ขึ้น GitHub Pages

  1. ไปที่ github.com/naadisak/itborrow
  2. Upload ไฟล์ index.html (และ setup.html ถ้าต้องการ)
  3. ไปที่ Settings → Pages
  4. Source: Deploy from a branch → branch main → folder / (root)
  5. กด Save — รอ 1-2 นาที จะได้ URL: https://naadisak.github.io/itborrow/
